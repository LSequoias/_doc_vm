Cas d'utilisation d'une Machine Virtuelle

Les machines virtuelles sont comme des containers fonctionnant indépendamment du sytème principale, les VM ont leurs propres adresse IP, leurs propres adresse physique générée selon la virtualisation, un firewall indépendant. Pour fonctionner une VM crée une partition d'un système d'explotation sur le disque de la machine physique, en gros 15Go sont allouer à une distribution sur le disque local pour faire fonctionner une instance. Les distributions peuvent être sous format ISO.

  • D'isolée un environement sur la machine
  • D'installée un OS de façon isolée de la machine physique
  • De pouvoir communiquée avec plusieurs VM pour des crash test par exemple
  • Migrée une VM vers la machine physique
  • Testée la compatibilité entre les différents OS pour une app.
